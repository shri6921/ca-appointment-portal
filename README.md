## 💻 Run This Project on Your Laptop (No Coding Required!)

Follow these simple steps — takes only **5 minutes**!

---

### ✅ STEP 1 — Install Python (Only once)

1. Go to 👉 **[https://www.python.org/downloads/](https://www.python.org/downloads/)**
2. Click the big yellow **"Download Python"** button
3. Open the downloaded file
4. ⚠️ **IMPORTANT:** Before clicking Install, **check the box** that says:
   > ✅ **Add Python to PATH**
5. Click **Install Now** and wait for it to finish

---

### ✅ STEP 2 — Download This Project

1. Click the green **"Code"** button on this GitHub page (top right area)
2. Click **"Download ZIP"**
3. Find the downloaded `.zip` file (usually in your **Downloads** folder)
4. **Right-click** on it → **"Extract All"** → Click **Extract**
5. Open the extracted folder

---

### ✅ STEP 3 — Start the Website

1. Inside the project folder, find the file called:
   > 📄 **`START_WEBSITE.bat`**
2. **Double-click** it
3. A black window will appear — wait a few seconds
4. Your browser will **automatically open** the website at `http://localhost:5000`

🎉 **That's it! The website is running on your laptop!**

## ✨ Features

- 📅 Book appointments with Chartered Accountants
- 📂 Secure Document Vault (upload Form 16, PAN, bank statements)
- 💳 Online Payment via Razorpay
- 🤖 AI Tax Assistant Bot (instant tax guidance)
- 📋 e-Sign & Document Verification
- 🔔 Real-time Notifications
- 🪪 PAN & GSTIN Registry
- 📊 Tax Compliance Calendar (ITR, GST, Advance Tax deadlines)
- 📄 Printable Payment Receipts
- 📱 Mobile & Laptop Friendly
